# Forms

## Text input

- Type: `text`

### Options

| Option | Description                                                   | Required |
| :----- | :------------------------------------------------------------ | :------- |
| type   | `text`                                                        | Yes      |
| label  | The label that describes the field                            | No       |
| rules  | Validation rules for the field (See [validation](Validation)) | No       |
